Hostel Management System

A web-based Hostel Management System built using React (Vite), Firebase, and Framer Motion.
It allows admins to manage hostlers, rooms, payments, and reports ‚Äî and hostlers to make payments, upload screenshots, and view payment status.

Tech Stack

Frontend:

React (Vite)

Tailwind CSS

Framer Motion

Backend / Database:

Firebase Firestore

Firebase Authentication

Firebase Hosting

Features

Hostler Registration & Room Allocation
Payment Upload with Screenshot
Automatic Payment Tracking & Due Date Handling
Admin Approval System
Payment & Revenue Reports
Responsive UI with Animations


 Environment Variables

Create a .env file in the root directory and add your Firebase configuration:

VITE_FIREBASE_API_KEY=your_api_key

VITE_FIREBASE_AUTH_DOMAIN=your_auth_domain

VITE_FIREBASE_PROJECT_ID=your_project_id

VITE_FIREBASE_STORAGE_BUCKET=your_storage_bucket

VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id

VITE_FIREBASE_APP_ID=your_app_id

VITE_FIREBASE_MEASUREMENT_ID=your_measurement_id



 Installation & Setup
# Clone the repo
git clone https://github.com/yourusername/hostel-management.git

# Go inside the folder
cd hostel-management

# Install dependencies
npm install

# Start the development server
npm run dev

Scripts
Command	Description
npm run dev	Start the Vite development server
npm run build	Build the app for production
npm run preview	Preview the production build locally
